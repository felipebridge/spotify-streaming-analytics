## 📌 Descripción del Proyecto

Proyecto de Data Analytics enfocado en el análisis de consumo musical utilizando Dataset de Spotify.

Se diseñó un pipeline ETL en Python, se modelaron los datos en PostgreSQL ejecutado mediante Docker y se construyó un dashboard analítico en Power BI para visualizar patrones de comportamiento, engagement y tendencias temporales.

---

## 🎯 Objetivos Analíticos

- Identificar los artistas y canciones más reproducidos.
- Analizar la evolución temporal del consumo.
- Calcular la tasa de saltos (abandono).
- Detectar patrones horarios de escucha.
- Evaluar diferencias de uso según dispositivo.
- Medir la intensidad de escucha.
